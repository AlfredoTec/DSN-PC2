# TechStore Frontend

UI en React + Vite + TailwindCSS + DaisyUI + Lucide Icons.

## Scripts
- npm run dev (http://localhost:5174)
- npm run build
- npm run preview

## Configuración
- Crear `.env` basado en `.env.example` si quieres apuntar a otra API: `VITE_API_BASE=https://api.tu-dominio.com`
- Por defecto, el proxy de Vite reenvía `/api` a `http://localhost:4000`.
